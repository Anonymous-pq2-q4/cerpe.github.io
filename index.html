Communication-Efficient Relative Pose Estimation with Vision Foundation Models for
Ephemeral Collaborative Perception

Paper Code

Abstract

Relative pose estimation is a fundamental capability for collaborative perception and coordination in multi-robot systems. However, robots encountering each other in real-world environments often operate in short interaction windows and must operate under limited communication bandwidth with intermittent or missing visual overlap caused by occlusions or limited fields of view. Existing approaches typically rely on global reference frames, assume sustained view overlap, or incur prohibitive communication costs, thereby limiting their applicability to ephemeral collaborative perception.

To address these challenges, we introduce communication-efficient relative pose estimation (CERPE) that integrates vision foundation models to jointly estimate ego-motion and inter-robot relative pose. Communication efficiency is achieved by decoupling overlap detection from pose estimation, using compact descriptors for continuous inter-robot communication and selectively exchanging key raw observations only during overlapping encounters. Non-overlapping encounters are handled by propagating inter-robot relative poses through metrically scaled ego-motion, thus maintaining relative pose estimates even in the absence of visual overlap. Experiments in simulation and real-world robots demonstrate that CERPE achieves robust 6-DoF pose estimation in ephemeral collaborative perception.

Approach

CERPE restructures vision foundation models to jointly estimate metrically scaled ego-motion and inter-robot relative poses under communication constraints. Each robot encodes its observations into compressed visual descriptors using SALAD and continuously shares them to detect visual overlap. When overlap is detected, raw observations are selectively exchanged, and VGGT estimates the inter-robot relative pose. To resolve monocular scale ambiguity, we align predicted depth with metric depth from Metric3Dv2. During non-overlapping periods, relative poses are propagated using metrically scaled ego-motion.

Formation control in indoor environments

A leader robot is teleoperated via keyboard, while follower robots autonomously maintain a fixed relative pose to the leader using CERPE's pose estimates.

Real World Experiment Details

Hardware Setup

We conduct experiments with teams of 3 and 5 Agilex Limo Pro robots in indoor environments. All onboard RGB and depth images are transmitted to a centralized laptop with RTX 4080 GPU via TCP with image compression. The per-robot communication bandwidth is approximately 60 KB per transmission. Onboard depth sensors provide metric depth for scale recovery.

Hierarchical Control Pipeline

We employ a two-level hierarchical control architecture to bridge CERPE's perception output with robot actuation:

  • High-level (1.5–4 Hz): CERPE takes RGB images from all robots and estimates the relative poses between the leader and each follower. From these estimates, each follower's goal position is computed. The inference latency is 250 ms (2 robots), 370 ms (3 robots), and 620 ms (5 robots).
  • Low-level (15–30 Hz): Each follower interpolates the goal position from CERPE and tracks it using an MPC controller with a bicycle kinematic model. Between CERPE updates, pose is maintained via wheel odometry.
